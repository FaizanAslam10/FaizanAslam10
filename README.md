## Featured Projects

### Qanoon - AI Legal Research Assistant

AI-powered legal research assistant for Pakistani law. It answers plain-language legal questions with relevant provisions, consequences, procedures, and advice, grounded in cited statute text.

- Built a RAG pipeline over 1,000+ Pakistani statutes
- Used vector embeddings and semantic search with a Postgres vector store
- Developed with Next.js and deployed on Vercel
- Live: https://legal-rag-assistant-two.vercel.app/

### TradeRank - Marketing Platform for Local Service Businesses

High-conversion marketing platform for a digital agency serving local trades.

- Built live-results dashboards, animated metrics, ranking tables, and lead-capture flows
- Optimized for SEO and performance with Next.js and Tailwind CSS
- Designed a terminal-inspired visual system for a distinctive brand experience

Final-year project converting 2D residential floor plans into interactive 3D walkthrough models.

- Built an automated pipeline from architectural plans to 3D visualization
